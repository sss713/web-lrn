2: Ветвление в Git

|*||*||*||*|

###
Узнал команды:
git checkout branchName # Переключение между ветками
git branch branchName # Создать ветку
git commit #Добавть комитт (записать внесенные изменения) на выбраной ветке
git merge branchName # Произвести слияние изменений с указанной веткой
git rebase branchName # Перенести все коммиты из указанной ветки в текущую

git checkout -b branchName # позволяет создать ветку и сразу переключиться на неё
